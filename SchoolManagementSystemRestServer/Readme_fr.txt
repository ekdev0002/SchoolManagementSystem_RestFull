Information
-------------------------------------------------------------------------------
API Rest d'accès aux données de l'application de gestion d'école.

Instructions
-------------------------------------------------------------------------------
1- Créer un vhost sur serveur web( apache + php)
2- Créer une base de donnée mysql nommée schoolsm
3- Importer le shemas de base de donnée dans le fichier schoolsm vers la base la base de donnée
4- copier le repertoire dans le dossier www du serveur web
5- configurer le repertoire dans la configuration du vhost
6- configuer les parametres de la base donnée dans le fichier src/configs/DBAccess.php
