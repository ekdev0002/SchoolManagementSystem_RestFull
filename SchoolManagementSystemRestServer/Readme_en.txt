Information
-------------------------------------------------------------------------------
Rest API accessing data  for  for the school management application.

Instructions
-------------------------------------------------- -----------------------------
1- Create a virtualhost (vhost) on web server (apache + php)
2- Create a MySQL database named schoolsm
3- Import the database shemas in the schoolsm.sql file to the database
4- Copy the directory to the www folder of the web server
5- Configure the directory in the vhost configuration
6- Configure the parameters of the database given in the src/configs/DBAccess.php file
